# Setup
## With `uv`

```bash
$ uv venv
$ uv syn --dev
$ uv run python3 -m ipykernel install --user --name=sciconf-2024
```

## With `pip`

```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python3 -m pip install -r requirements.txt
$ python3 -m ipykernel install --user --name=sciconf-2024
```
